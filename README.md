# IABoat
Le projet contient les différents codes du PIC 2025-2026 qui vont être utilisés sur le bateau IA.   

# Instructions pour commencer à travailler :   

## Récupérer les fichiers
1 - Se créer un compte Github.   
2 - Donner son identifiant à Thomas pour qu'il vous rajoute comme contributeur.   
3 - Créer sa clé ssh sur son PC. On travaille en linux donc installez une VM si besoin (genre WSL). Voir tutoriel ici pour la génération de clés : https://docs.github.com/fr/authentication/connecting-to-github-with-ssh/generating-a-new-ssh-key-and-adding-it-to-the-ssh-agent   
4 - Si pas déjà fait en suivant le tutoriel au dessus, ajouter votre clé à votre compte Github.   
5 - Si tout fonctionne correctement vous aller pouvoir cloner le repo `git clone git@github.com:TameoMEBC/IABoat.git`

Suivre ensuite les instructions spécifiques aux sous-dossiers
