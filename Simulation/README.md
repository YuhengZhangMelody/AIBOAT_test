# Simulation-2026-PIC
Le projet contient la simulation qui va, a terme, servir pour entrainer l'algo de path planning en reinforcement learning.   

# Instructions pour commencer à travailler :   

## Récupérer les fichiers
Suivre les instructions du readme précédent

## Créer l'environnement virtuel
1 - Installer python 3.12.X avec X la dernière version   
2 - Créer l'environnement virtuel avec `python -m venv <env_name>` (Vous pouvez vérifier la version installée avec `python --version`)
3 - Activer votre environnement fraichement crée : `source pathtomyenv/bin/activate`. (Si l'environnement est bien activé, son nom apparaitra en début de ligne de commande.)   
4 - Installer ensuite les packets nécessaires : `pip install -r requirements.txt`   

## Lancer le code
1 - Se placer dans le dossier "Simulation-2026-PIC"   
2 - Activer votre environnement virtuel `python -m venv <env_name>`    
3 - Taper `python Sousdossier/mon_fichier.py`
